About The Project

What started as a desktop graph-based navigator Python application is now a modern, web-ready portfolio project. PathFinder takes complex graph theory—specifically Dijkstra's Algorithm—and applies it to a real-world map visualization. It completely runs statically in the browser, providing quick responses without needing a backend server.

Key Features

⚡ Dijkstra's Algorithm

Calculates the absolute shortest path using customized node weights reflecting realistic travel effort.

🗺️ Real Geographic Tracking

Powered by Leaflet.js and OpenStreetMap, integrating abstract graph logic into real latitude and longitude positions.

💻 Zero Backend

Fully client-side operations. Computation is performed completely within your browser using JavaScript.
